## 📌 Project Overview

A web-based **Student Management System** built using Django that allows users to manage student records efficiently with CRUD operations.

---

## ✨ Features

* ➕ Add Student
* 📋 View Student List
* ✏️ Update Student
* ❌ Delete Student
* 🔐 Admin Panel
* 📅 Date Picker for DOB
* 🎨 Responsive UI with Bootstrap

## ⚙️ Installation

```id="8n4szq"
git clone https://github.com/your-username/student-management.git
cd student-management
python -m venv myenv
myenv\Scripts\activate
pip install django
python manage.py migrate
python manage.py runserver
```
